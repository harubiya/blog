https://docs.flutter.dev/deployment/android

1. Adding a launcher icon
- flutter pub add flutter_launcher_icons
- 아이콘 만들기
- pubspec.yalm 추가
- dart run flutter_launcher_icons

2. Create an upload keystore (프로젝트 시작할때 이전 파일들 복사해두자)
- keytool -genkey -v -keystore ~/appkey/upload-keystore.jks -storetype JKS -keyalg RSA -keysize 2048 -validity 10000 -alias myalias
- upload-keystore.jks 파일을 andorid/app/ 밑에 복사
- android/key.properties 만들기(복사)
- android/app/build.gradle 수정(복사)

===============
(소스 정리: print 지우기, debug를 release로 수정하기)
3. 이전 빌드 apk, appbundle 파일 따로 저장
4. pubspec.yalm version 수정 (1.0.0+1) +숫자(버전코드) 올리기
5. flutter clean
6. flutter build apk
7. flutter install (마지막 테스트)
8. flutter build appbundle
9. 스토어에 .aab file / update 올리기
