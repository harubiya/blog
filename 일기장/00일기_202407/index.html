지구환경에는 온디바이스가 더 나을까? /////chagpt `sed`(stream editor)는 텍스트 파일을 조작하는 데 사용되는 강력한 도구입니다. 주로 파일의 내용을 검색하고 변환하거나 특정 패턴을 치환하는 작업에 사용됩니다. ### 기본 사용법 `sed`의 기본 형식은 다음과 같습니다: ```bash sed 's/old/new/' file ``` 이 명령어는 `file`에서 `old`라는 패턴을 찾아 `new`로 한 번만 바꿉니다. ### 주요 옵션 및 예제 3. **전체 라인 치환** ```bash sed 's/World/Universe/g' input.txt ``` `g` 플래그를 사용하여 한 라인의 모든 `World`를 `Universe`로 치환합니다. 4. **파일 수정** ```bash sed -i 's/World/Universe/g' input.txt ``` `-i` 옵션을 사용하여 `input.txt` 파일을 직접 수정합니다. 5. **특정 라인 치환** ```bash sed '2s/World/Universe/' input.txt ``` `input.txt` 파일의 두 번째 라인에서만 `World`를 `Universe`로 치환합니다. 7. **패턴으로 라인 삭제** ```bash sed '/pattern/d' input.txt ``` `input.txt` 파일에서 `pattern`을 포함하는 모든 라인을 삭제합니다. 8. **라인 번호로 라인 삭제** ```bash sed '2d' input.txt ``` `input.txt` 파일의 두 번째 라인을 삭제합니다. 9. **라인 삽입** ```bash sed '2i\This is a new line.' input.txt ``` `input.txt` 파일의 두 번째 라인 위에 `This is a new line.`을 삽입합니다. 10. **특정 라인 뒤에 라인 추가** ```bash sed '2a\This is a new line.' input.txt ``` `input.txt` 파일의 두 번째 라인 뒤에 `This is a new line.`을 추가합니다. ### 고급 예제 1. **범위 내 치환** ```bash sed '2,4s/World/Universe/' input.txt ``` `input.txt` 파일의 2번째부터 4번째 라인 사이에서 `World`를 `Universe`로 치환합니다. 2. **정규 표현식 사용** ```bash echo "2024-07-12" | sed 's/\([0-9]\{4\}\)-\([0-9]\{2\}\)-\([0-9]\{2\}\)/\3\/\2\/\1/' ``` 출력: `12/07/2024` 이 명령어는 날짜 형식을 `YYYY-MM-DD`에서 `DD/MM/YYYY`로 변환합니다. 3. **파일 끝에 라인 추가** ```bash sed '$a\This is the end of the file.' input.txt ``` `input.txt` 파일의 마지막 라인에 `This is the end of the file.`을 추가합니다. `sed`는 매우 유연하고 강력한 도구로, 다양한 텍스트 조작 작업을 자동화하는 데 유용합니다. /////////////////////////////////////// 2024.7.11 - 다른도서관 첫방문 다니던 도서관 말고, 새로 지은 도서관 처음 가봄. 1년은 됐지?
